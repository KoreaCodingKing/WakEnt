꾸준히 왁타버스 유튜브 컨텐츠 영상 업데이트하고 있습니다.

## 개발 방법

저장소를 클론한 후 yarn 명령어를 사용하여 패키지를 다운받아주세요. yarn이 없는 경우에는 `npm install -g yarn` 으로 설치할 수 있습니다.

```sh
yarn
```

그 후 다음 명령어를 통해 서버 실행, 빌드를 할 수 있습니다.

```sh 
yarn dev # 개발 서버 실행
yarn start # 서버 실행
yarn build # 빌드
yarn lint # 린트 체크
```
